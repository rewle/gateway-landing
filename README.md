# gateway-landing

Публичный маркетинговый лендинг Gateway. Статика без сборки — vanilla HTML/CSS/JS, как и `src/frontend` в соседнем репозитории.

## Запуск локально

```bash
python3 -m http.server 8899
# → http://localhost:8899
```

Правки в `index.html`/`styles.css`/`script.js` видны сразу по обновлению страницы, шаг сборки не нужен.

## Структура

```
index.html   # разметка, весь копирайт
styles.css   # токены (цвет/типографика), секции, адаптив
script.js    # редактирование полей в hero-карточке, скролл-reveal, вкладки тарифов
```

## Источник контента

Копирайт и структура секций взяты из `~/Projects/mind/Проекты/Gateway/{Gateway.md,Слайды.md}` (проблема/решение/как работает/возможности/тарифы) — это внутренние заметки, не публикуются напрямую.

**Сознательно не включено** (материал инвесторского питча, не для публичного сайта):
- Сумма раунда, доля в компании, финмодель/ARR
- Информация о найме команды, поиске операционного директора
- Детальный конкурентный анализ (рвы/риски) — на сайте только сжатое сравнение с альтернативами
- Реальный IP demo-стенда — вместо прямой ссылки на публично торчащий наружу инстанс с дефолтными учётками используется CTA-форма/почта; когда появится отдельный демо-контур с одноразовыми учётками, можно добавить прямую ссылку

## Дизайн

Визуальная метафора — маскирование/редактирование полей (ключевая механика продукта): чёрные полосы поверх чувствительных значений, штамп «проверено», типографика в духе дела/досье (IBM Plex Mono для заголовков и лейблов, IBM Plex Sans для текста — обе с кириллицей).
